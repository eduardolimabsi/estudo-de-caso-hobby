# Projeto: Melhores Filmes de 2025

Este é um pequeno aplicativo em Vue que criei para treinar e mostrar como portfólio. O objetivo inicial dele é listar alguns dos filmes mais aguardados para o começo de 2025 e ter algumas páginas de detalhamento, usando o Vue Router para trocar a tela sem recarregar a página (estilo SPA).

## Como rodar no seu computador

1. Primeiro você precisa instalar as dependências do projeto. Abra o terminal na pasta raiz e digite:
```bash
npm install
```

2. Em seguida, para rodar o modo de desenvolvedor, execute:
```bash
npm run dev
```

Pronto! É só abrir o link que aparecer no terminal e testar a galeria.
